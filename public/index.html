Cara Main & Fitur

  1. Login atau daftar dengan username + PIN.
  2. Buat room atau join dari Public Lobby.
  3. Host klik Mulai Game saat pemain minimal 2.
  4. Samakan warna, angka, atau simbol. Wild bebas pilih warna.
  5. Tekan UNO saat kartu tinggal 1. Pemain lain bisa Challenge kalau lupa.
  6. Shop untuk beli skin/power. Power bersifat sekali pakai dan stok akan berkurang saat digunakan.
  7. Crates memberi reward acak. Inventory untuk equip skin, frame, badge, dan tema.
  8. Friends untuk add teman dan invite ke lobby.
  9. Host bisa cari lagu YouTube di tombol musik melayang, lalu Play Room agar didengar bersama.
  10. Join Voice untuk ngobrol; daftar pemain voice akan terlihat.
